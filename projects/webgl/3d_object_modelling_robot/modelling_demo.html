  - 'K' and 'L' keys to rotate the Robot's hand;

  - Left Arrow (←) key to move the Robot to the left. And Right Arrow (→) key to move the Robot to the right;

  - Up Arrow (↑) key to move the Robot forward. And Down Arrow (↓) key to move Robot backward;
